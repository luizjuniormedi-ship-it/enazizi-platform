

## Publicar como PWA

O app **já está 100% configurado como PWA** — manifest, service worker, ícones e página `/install` já existem. Não é necessária nenhuma mudança de código.

### Para publicar agora:

1. Clique no botão **"Publish"** no canto superior direito
2. Clique em **"Update"** para publicar
3. O app ficará acessível em: **https://study-buddy-ai-560.lovable.app**

### Para os usuários instalarem no celular:

Compartilhe o link do app. Os usuários podem:
- **Android**: Menu do navegador (⋮) → "Instalar app"
- **iPhone**: Botão Compartilhar → "Adicionar à Tela de Início"
- Ou acessar **https://study-buddy-ai-560.lovable.app/install** para ver instruções detalhadas

Não há código para implementar — basta publicar clicando no botão.
